## Instalação 🛠️

Para executar o projeto em sua própria máquina, siga estas etapas:

1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Inicie o projeto com `npm start`.

### Acesso Admin e Usuário de Exemplo
